# Rust 文字列イテレータチートシート

## 文字列のイテレータを作る

```rust
let s = "hello";

s.chars()          // 文字のイテレータ
s.split_whitespace()  // 空白で分割したイテレータ
s.split(',')       // カンマで分割したイテレータ
```

## String の型による参照の書き方

### Vec<char>

```rust
let v = vec!['a', 'b', 'c'];
v.iter().filter(|&&c| c.is_alphabetic())
//              ^^ char の参照の参照を外す
```

### &str

```rust
let s = "hello";
s.chars().filter(|c| c.is_alphabetic())
//        ^ char は参照ではない
```
