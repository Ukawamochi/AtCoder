# 型についてチートシート

## 整数型

| 型 | ビット数 | 範囲 | 用途 |
|-----|---------|------|------|
| `i32` | 32 | -2^31 ～ 2^31-1 | 整数 |
| `i64` | 64 | -2^63 ～ 2^63-1 | 整数 |
| `isize` | 64(32) | -2^63 ～ 2^63-1 | ? |
| `u32` | 32 | 0 ～ 2^32-1 | 非負数 |
| `u64` | 64 | 0 ～ 2^64-1 | 非負数 |
| `usize` | 64(32) | 0 ～ 2^64-1 | 配列 |

## 浮動小数点数型

| 型 | ビット数 |
|-----|---------|
| `f32` | 32 |
| `f64` | 64 |

## 文字・文字列型

| 型 | 説明 |
|-----|------|
| `char` | 1文字（例：`'a'`） |
| `&str` | 文字列スライス（例：`"hello"`） |
| `String` | 所有する文字列 |

## コレクション型

| 型 | 説明 | 例 |
|-----|------|-----|
| `Vec<T>` | 可変長配列 | `vec![1, 2, 3]` |
| `[T; n]` | 固定長配列 | `[1, 2, 3]` |
| `HashMap<K, V>` | キーと値のペア | `HashMap::new()` |
| `HashSet<T>` | 重複なしの集合 | `HashSet::new()` |

## ブール値型

| 型 | 値 |
|-----|-----|
| `bool` | `true` / `false` |
