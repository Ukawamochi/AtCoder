# Rust 数値型イテレータチートシート

### Vec からイテレータを作る

```rust
let v = vec![1, 2, 3];

v.iter()           // 参照のイテレータ（v は使用可能）
v.iter_mut()       // 可変参照のイテレータ（要素を変更可能、v は使用可能）
v.into_iter()      // 所有権のイテレータ（v の所有権を奪う、v は使用不可）
```
## 基本的な操作
### 追加
- push()
- insert()
## 削除
- remove(index)

### 取得
- first()
- last()

## 数値のイテレータメソッド

- map()
- filter()
- fold()
- sum()
- any()
- all()
- find()
- take()
- skip()
- rev()
- enumerate()


#### `map(|x| 式)`：各要素に処理を適用

```rust
let v = vec![1, 2, 3];
v.iter().map(|x| x * 2)
// 処理の流れ：1 → 1*2, 2 → 2*2, 3 → 3*2
```

実行例：
```rust
let v = vec![1, 2, 3];
let result = v.iter().map(|x| x * 2).collect::<Vec<_>>();
// result は [2, 4, 6]
```

#### `filter(|x| 条件)`：条件に合う要素のみを通す

```rust
let v = vec![1, 2, 3, 4, 5];
v.iter().filter(|&&x| x > 2)
// 処理の流れ：1（除外）, 2（除外）, 3, 4, 5
```

実行例：
```rust
let v = vec![1, 2, 3, 4, 5];
let result = v.iter().filter(|&&x| x > 2).collect::<Vec<_>>();
// result は [3, 4, 5]
```

#### `fold(初期値, |累積, 要素| 式)`：累積計算を実行

```rust
let v = vec![1, 2, 3, 4];
v.iter().fold(0, |acc, x| acc + x)
// 処理：0 + 1 = 1, 1 + 2 = 3, 3 + 3 = 6, 6 + 4 = 10
// 結果：10
```

実行例：
```rust
let v = vec![1, 2, 3, 4];
let sum = v.iter().fold(0, |acc, x| acc + x);
// sum は 10

let product = v.iter().fold(1, |acc, x| acc * x);
// product は 24
```

#### `sum::<型>()`：すべての要素を足す

```rust
let v = vec![1, 2, 3, 4];
let sum: i32 = v.iter().sum();
// sum は 10
```

#### `any(|x| 条件)`：1つでも条件を満たす要素があるか

```rust
let v = vec![1, 2, 3, 4, 5];
v.iter().any(|&&x| x > 3)
// 結果：true（4と5が条件を満たす）
```

#### `all(|x| 条件)`：すべての要素が条件を満たすか

```rust
let v = vec![1, 2, 3, 4, 5];
v.iter().all(|&&x| x > 0)
// 結果：true（すべて0より大きい）
```

#### `find(|x| 条件)`：条件を満たす最初の要素を返す

```rust
let v = vec![1, 2, 3, 4, 5];
let result = v.iter().find(|&&x| x > 3);
// result は Some(&4)
```

#### `take(n)`：最初の n 個の要素を取得

```rust
let v = vec![1, 2, 3, 4, 5];
v.iter().take(3)
// 処理の流れ：1, 2, 3（4, 5は除外）
```

実行例：
```rust
let v = vec![1, 2, 3, 4, 5];
let result = v.iter().take(3).collect::<Vec<_>>();
// result は [1, 2, 3]
```

#### `skip(n)`：最初の n 個の要素を処理の対象から外す

```rust
let v = vec![1, 2, 3, 4, 5];
v.iter().skip(2)
// 処理の流れ：3, 4, 5（1, 2は除外）
```

実行例：
```rust
let v = vec![1, 2, 3, 4, 5];
let result = v.iter().skip(2).collect::<Vec<_>>();
// result は [3, 4, 5]
```


#### `rev()`：逆順にする

```rust
let v = vec![1, 2, 3];
v.iter().rev()
// 処理の流れ：3, 2, 1
```

実行例：
```rust
let v = vec![1, 2, 3];
let result = v.iter().rev().collect::<Vec<_>>();
// result は [3, 2, 1]
```

#### `enumerate()`：インデックス付きでループ

```rust
let v = vec!['a', 'b', 'c'];
v.iter().enumerate()
// 処理の流れ：(0, 'a'), (1, 'b'), (2, 'c')
```

実行例：
```rust
let v = vec!['a', 'b', 'c'];
for (i, &c) in v.iter().enumerate() {
    println!("{}: {}", i, c);
}
// 0: a
// 1: b
// 2: c
```

## イテレータの実行（終端操作）

イテレータは以下のメソッドで初めて実行されます。

```rust
.collect::<Vec<_>>()  // ベクトルに集める
.sum::<i32>()         // 合計を計算
.count()              // 要素数をカウント
.find(条件)           // 条件を満たす要素を探す
.any(条件)            // 条件を満たす要素があるか
.all(条件)            // すべてが条件を満たすか
```

## 複数の処理を組み合わせる

```rust
let v = vec![1, 2, 3, 4, 5, 6];

let result = v.iter()
    .filter(|&&x| x % 2 == 0)     // 偶数のみ
    .map(|x| x * x)                // 二乗
    .collect::<Vec<_>>();

// 処理の流れ：
// [1, 2, 3, 4, 5, 6]
// → filter：[2, 4, 6]
// → map：[4, 16, 36]
// → collect：[4, 16, 36]
```

## Vecの型による参照の書き方

### Vec<i32>
```rust
let v = vec![1, 2, 3];
v.iter().filter(|&&x| x > 1)
//              ^^ i32 の参照の参照を外す
```
