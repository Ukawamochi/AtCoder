# 型変換チートシート


### すべての数値型 → String

```rust
let n = 123;
let s = n.to_string();
```

### すべての数値型 → char(ASCII)

```rust
let n = 65;
let c = n as char;  // 'A'
```

### char → 数値
(代入先の変数の型に依存)

```rust
let c = '5';
//10進数でu32に変換
let n: u32 = c.to_digit(10).unwrap();
```

### 文字列 → 数値型
(代入先の変数の型に依存)

```rust
let s = "123";
//ここではi32に変換される
let n: i32 = s.parse().unwrap();
```

### 数値型 → 数値型
数値型を別の数値型に変換する

```rust
let x: i32 = 10;
let y: i64 = x as i64;       // i32 → i64
let z: f64 = x as f64;       // i32 → f64
let a: i32 = (10.5 as i32);  // f64 → i32（小数部は切り捨て）
```


## 型のメソッド

### Vec<T>

```rust
v.push(要素)        // 要素を追加
v.pop()             // 最後の要素を削除
v.len()             // 要素数
v.sort()            // 昇順ソート(降順にしたいなら.reverse()を追加)
v.reverse()         // 前後を反転
```

#### sort()メゾットの使い方
クイックソートとヒープソートの組み合わせでソートされる。(イントロソート)

```rust
    // i32
    let mut v = vec![3, 1, 4];
    v.sort();  // [1, 3, 4]
    
    // f64のときはsort_byを使う(sortは使用不可)
    let mut v = vec![3.5, 1.2, 4.1];
    v.sort_by(|a, b| a.partial_cmp(b).unwrap());  // [1.2, 3.5, 4.1]
    
    //ダブルのVecのソート
    let mut v: Vec<(i64, i64)> = Vec::new();
    v.push((1, 2));
    v.sort_by(|a, b| a[0].cmp(&b[0]));//昇順でソート


    
    // String
    let mut v = vec!["c", "a", "b"];
    v.sort();  // ["a", "b", "c"]
    
    // char
    let mut v = vec!['c', 'a', 'b'];
    v.sort();  // ['a', 'b', 'c']
```

### String / &str

```rust
s.len()              // 長さ
s.split(区切り文字)  // 分割

//型変換
s.to_string()        // String に変換
s.parse()            // 数値に変換
```

### char to char

```rust
c.is_alphabetic()    // 英字か判定
c.is_numeric()       // 数字か判定
c.to_uppercase()     // 大文字に変換
c.to_lowercase()     // 小文字に変換
    
//型変換
c.to_digit(10)       // 数値に変換
```

## 宣言と初期化

```rust
// ベクトル
let v = vec![1, 2, 3];
let mut v: Vec<i32> = Vec::new();

// 固定長配列
let arr = [1, 2, 3];
let arr: [i32; 3] = [1, 2, 3];
```
